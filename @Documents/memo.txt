PetaPoco4Core
	こいつのプロジェクトは .NET Standard 2.1ってなっているけど、
	ソース自体は .NET Framework 4.6.xでもコンパイルできるようにする。
	実際に仕事で使うときは、これらのソースファイルをコピって使うので。
	（コンパイル済みのDLLを参照するのではなくて）

-- ----------------------------------------------------------------------------
.NET Core SDK 3.1以上が必要（.NET Core Test Explorerが必要とする）

-- ----------------------------------------------------------------------------
IMapper と DefaultMapper
	カスタムなマッピングをこの型で作成して、Database.Mapperプロパティにセットしておくことで、
	マッピングをカスタマイズできる。

	でもver4.7.2で PocoData.GetConverter を修正したので、それまではOracleとSQLiteで
	うまくいかなかったcastができるようになったので、今の所不要。

-- ----------------------------------------------------------------------------

PetaPocoAppFx
	これは.NET Framework 4.6.1の動作の確認用コンソールアプリです。
